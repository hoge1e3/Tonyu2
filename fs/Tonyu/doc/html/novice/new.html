複数のキャラクタを配置しましょう

さて，Appleを実行すると，リンゴが表示されますが，猫は出てこなくなってしまいました．ゲームには，猫とリンゴが同時に出てくる必要があります．

そこで「リンゴと猫を置く」ための別のプログラムを作りましょう．

  • メニューの「ファイル」→「新規」を選びます
  • ファイル名を入力します
    • ここでは Game と入力してみます

Gameに，次のように入力してみましょう．

Game
new Cat;
new Apple;

実行メニューから，「Gameを実行」選びましょう． すると，猫とリンゴが同じ画面に表示されます．

ここで出てきたnew という命令は， 新しくキャラクタを作るための命令です．

次のように，プログラム名を指定します． 新しく出現したキャラクタは， 指定されたプログラム名のプログラムを実行します． 

new プログラム名;

なお，今後はしばらく Game を実行していきますので「実行する」と書いてあったら， 実行メニューから，「Gameを実行」選ぶようにしてください． F9キーを押すと，前回実行したプログラムと同じプログラムが実行されるので便利です．
