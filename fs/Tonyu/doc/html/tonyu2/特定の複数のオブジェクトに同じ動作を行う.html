オブジェクトを表示する

特定の複数のオブジェクトに同じ動作を行う

Main
new Chara1{x:100, y:100};
new Chara1{x:200, y:100};
new Chara2{x:300, y:100};
while(true){
    for (c in all(Chara1)) {
        c.x++;
        c.y++;
    }
    update();
}
