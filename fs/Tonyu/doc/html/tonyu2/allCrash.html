BaseActor

allCrashメソッド

指定されたクラスのオブジェクトのうち，自分と衝突しているすべてのオブジェクトをあらわすTQueryオブジェクトを返します．

書式

allCrash(Class)

Classで指定されたクラスのオブジェクトのうち，自分と衝突しているすべてのオブジェクトをあらわすTQueryオブジェクトを返します．

例

    // このオブジェクトがTargetクラスのオブジェクトとぶつかっていたら，
    // そのオブジェクトを消す
    for (t in allCrash(Target)) {
        t.die();
    }

上の例は，次のように書くこともできます． 

    // このオブジェクトがTargetクラスのオブジェクトとぶつかっていたら，
    // そのオブジェクトを消す
    allCrash(Target).die();
