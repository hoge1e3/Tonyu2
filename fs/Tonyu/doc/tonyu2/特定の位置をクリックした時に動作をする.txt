[[オブジェクトを動かす]]

*特定の位置をクリック/タッチした時に動作をする
<<code
x=100;y=200;
while(true){
  t=$touches[0];
  if(t.touched==1 && dist(x-t.x,y-t.y)<20){
     x+=10;
  }
  update();
}
>>
タッチする（またはマウスをクリック）し、このオブジェクトとマウスカーソルとの距離が20未満の（オブジェクトの中心から半径20以内の円をタッチされた）とき、横に移動する。

タッチされたかどうかの判定を円ではなく四角形で行なうには，次のようにする：

<<code
x=100;y=200;
while(true){
  t=$touches[0];
  if(t.touched==1 && abs(t.x-x)<20 && abs(t.y-y)<20  ){
     x+=10;
  }
  update();
}
>>
