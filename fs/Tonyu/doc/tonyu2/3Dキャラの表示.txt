*・3Dキャラの表示

3Dの計算を関数化する

Mainファイルを作る

ThreeDファイルを作る

<<codeThreeD
function threeD(){
    hs=50*200/(50+z); /大きさの計算
    scaleX = hs;　　　//大きさの代入
    
    y=oy*50/(50+z)+$screenHeight/2;　
    x=ox*50/(50+z)+$screenWidth/2;　 
    zOrder=z;

　　//ox,oy,zを使いx,yの位置を計算して代入する

}
>>

ox,oy,zを[[とある計算>とある計算]]にあてはめ値を変化させるとあたかも3Dの世界にいるかのようにキャラが飛び出してくる。

Chara2ファイルを作る

<<codeChara2
 extends ThreeD;
while(true){   
    threeD();  
 
    z-=500;  
 
    if(z<0){
        die();
    }   
    update(); 
}
>>

キャラがプレイヤーに向かってくる処理を行う。

zを-2000ずつ行う事で向こうから手前に向かってくる。

そして画面と重なったら消す処理を行っている。

移動させたいキャラのファイルに上記のように書き込む。

これだけではエラーが起きるため[[3Dキャラの出現]]でMainフォルダに書き込んでいく。
