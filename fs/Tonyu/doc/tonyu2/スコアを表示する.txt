[[文字をテキストオブジェクトとして表示する]]

*スコアを表示する

やり方１

<<code Main
$score=0;
new Score;
while(true){
  if(getkey("z")==1) $score+=10;
  update();
}
>>
<<code Score
x=200;
y=200;
while(true) {
  text="Score:"+$score;
  update();
}
>>

やり方２

<<code Main
$score=0;
$scoreLabel=new Label{x:200,y:300};
while(true){
  if(getkey("z")==1) $score+=10;
  $scoreLabel.text="Score : "+$score;
  update();
}
>>

<<code Label
中身は何も書かなくてよい
>>

やり方１とやり方２は、二つともzキーを押すとscoreが10増える。
