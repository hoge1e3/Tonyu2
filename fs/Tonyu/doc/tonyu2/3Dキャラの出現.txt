**・3Dキャラクターの出現



<<codeMain
while(true){
　　//キャラ出現率10％
    if (rnd(100)<10) {
        new Chara2{ ox:rnd(100000)-50000,oy:rnd(100000)-50000,z:100000 ,p:1,zOrder:1};
    }
    update();
}
>>

3D表示を行うためプレイヤーから物体までの距離を設定していく。

ｚはキャラまでの距離を表しており、今回では100000としている。

ox,oyは出現位置です。
-なぜ100000や50000などの大きい値を設定しているかというと、初期位置の変化をわかりやすくするためです。

--近くの物体が１ｍ動くと動いたことはすぐにわかるが遠くの物体が１ｍ動いた場合はどうだろう？動いた気がする程度の物だろう。しかし遠くの物でも１００ｍ動いた場合はある程度はっきりわかることでしょう.

-今回のキャラは100000先に出現したため、大きく場所の変動を行っている。
