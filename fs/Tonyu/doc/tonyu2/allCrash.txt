[[BaseActor]]

* allCrashメソッド

指定されたクラスのオブジェクトのうち，自分と衝突しているすべてのオブジェクトをあらわす[[TQuery]]オブジェクトを返します．

**書式
<<code
allCrash(Class)
>>

[[@arg Class]]で指定されたクラスのオブジェクトのうち，自分と衝突しているすべてのオブジェクトをあらわす[[TQuery]]オブジェクトを返します．


*例

<<code
    // このオブジェクトがTargetクラスのオブジェクトとぶつかっていたら，
    // そのオブジェクトを消す
    for (t in allCrash(Target)) {
        t.die();
    }
>>

上の例は，次のように書くこともできます．

<<code
    // このオブジェクトがTargetクラスのオブジェクトとぶつかっていたら，
    // そのオブジェクトを消す
    allCrash(Target).die();
>>
