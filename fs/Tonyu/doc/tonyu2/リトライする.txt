[[すべてのオブジェクトを消す]]
*Main全体をwhileで囲う
<<code
while(true){
  //x,yなどの初期設定
  while(true){
    //ゲームの中身
    if(/*ゲームオーバーになったら*/){
      all().die();
      break;
    }
  }
  //ゲームの中身ここまで
}
>>

プログラム全体をループさせる事で、ゲームオーバーになった時にすべてのオブジェクトを消してゲームのループをbreak、再度ゲームを実行してゲームを繰り返しプレイする事が出来る。
