[[オブジェクトを表示する]]
*オブジェクトに動きを追加する
<<code
x=100;
y=100;
while(true){
  x++;
  update();
}
>>
オブジェクトの座標の値を加算することでそれが反映される。ここではxを加算し続けることで右に動くオブジェクトを作成している。

<<code
x=100;
y=100;
vy=0;
while(true){
  x++;
  y+=vy;
  vy+=0.1;
  if(y>$screenHeight){
    y=$screenHeight;
    vy=-vy;
  }
  update();
}
>>
こちらではvyというy方向の加速度を設定し、yに加えることで右に動きながら画面下部に触れると跳ねるオブジェクトを作成している。
