[[BaseActor]]

*getCrashRect

当たり判定に使う矩形領域を返します．
[[crashTo]]，[[crashTo1]]，[[allCrash]]で用いられます

**書式

<<code
getCrashRect()
>>

このオブジェクトの当たり判定に使う矩形領域を返します．

戻り値を[[@arg r]]とすると：
-([[@arg r]].x,[[@arg r]].y)が矩形領域の中心位置です（左上ではありません）
-[[@arg r]].widthと[[@arg r]].heightが，矩形領域の幅と高さです．

**オーバライド

デフォルトのgetCrashRectは，画像の大きさを基準に当たり判定を計算しますが，
getCrashRectをオーバーライドすると，当たり判定の大きさを変更できます．

[[@plistref chg]]の例では，当たり判定の大きさを5*5に設定しています．

<<code ChangeCrashRect.tonyu chg
\getCrashRect() {
   return {x,y,width:5, height:5};
}
>>
