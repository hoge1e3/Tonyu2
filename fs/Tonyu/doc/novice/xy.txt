*画像を縦や斜めにも動かしてみましょう

今まで，猫の画像は横にしか動きませんでしたが，縦にも動かすことができます．

<<code Cat y
y=50;
while (true) {
  y+=10;
  go(100,y);sleep();
}
>>

実行してみると，猫の画像が上から下に移動します．

さらに，横と縦に同時に動かすこともできます

<<code Cat xy
y=50;
x=100;
while (true) {
  y+=10;
  x+=10;
  go(x,y);sleep();
}
>>

実行してみると，猫の画像が左上から右下に斜めに移動します．

[[@plistref xy]]のように，
変数は同時に2つ使うこともできます．

変数を区別するために，それぞれの変数には名前が必要になります．ここでは x と y 
という名前の変数を使っています．

名前は，半角英字とアンダースコア(_)が使えます．2文字以上でも構いません．2文字目以降は数字も使うことができます．
