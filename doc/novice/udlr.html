画像をキーボードで上下左右に動かしましょう

さきほどのキーボードを使って右に動かす仕組みを使って， 画像を上下左右に動かしましょう

リスト1 Cat
x=50;
y=100;
while(true) {
   k=getkey("right");
   if (k>0) {
      x+=10;
   }
   k=getkey("left");
   if (k>0) {
      x-=10;
   }
   k=getkey("down");
   if (k>0) {
      y+=10;
   }
   k=getkey("up");
   if (k>0) {
      y-=10;
   }
   go(x,y);sleep();
}
