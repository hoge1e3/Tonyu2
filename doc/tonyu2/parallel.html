parallelメソッド

新しくスレッドを起動し、並行処理を行います

書式

parallel(methodName, args...)
  • methodName に、新しいスレッドが実行するメソッド名を指定します。指定されるメソッドは待機可能メソッドである必要があります。
  • args （省略可）に、メソッドに渡す引数を指定します

実行例 / ソース

戻り値

生成されたスレッド

補足

  • parallelメソッドで指定されたメソッドは待機可能モードで実行されます。
  • parallelメソッド自身は待機不能モードであっても呼び出すことができます。 

// objオブジェクトのfooメソッドを待機可能モードで実行
obj.parallel("foo"); // この呼出自身は待機不能モードで呼び出される
