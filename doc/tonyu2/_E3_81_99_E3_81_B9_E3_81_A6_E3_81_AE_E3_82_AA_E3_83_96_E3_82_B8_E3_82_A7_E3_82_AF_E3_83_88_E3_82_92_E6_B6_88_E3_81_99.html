new Enemy{x:100, y:200, p:4};
new Enemy{x:100, y:300, p:4};
while(true){
    if(getkey("space")==1){
        all().die();
        die();
    }
    update();
}
Enemy
何も書かなくてよい
スペースキーを押すとすべてのオブジェクトが消える。

※all().die()...自身以外のオブジェクトをすべて消す。

すべてのオブジェクトを消すのを利用すればゲームを繰り返し遊ぶプログラムなども実現できる。
