crashToメソッド

他のオブジェクトと衝突しているかを判定します．

書式1

crashTo(obj)

このオブジェクトがobjと衝突していればtrue，そうでなければfalseを返します．

書式2

crashTo(Class)

このオブジェクトがClassで指定されるクラスのオブジェクトのうちどれかと衝突していれば，そのうちどれか1つのオブジェクトを返します．そうでなければnullを返します

例

// $playerというオブジェクトにぶつかっていたら，$playerを消します
if (crashTo($player)) {
   $player.die();
}
