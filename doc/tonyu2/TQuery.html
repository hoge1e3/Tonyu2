TQuery

allメソッド， allCrashメソッドなどで返されるオブジェクトです．複数のActorに対して一斉に動作を行わせることができます．

要素数・要素へのアクセス

TQueryオブジェクトに格納しているActorの個数は.lengthで取得します．

各Actorへは[添字] でアクセスします．

例

a=all(Enemy);
print("敵の数=", a.length);
if (a.length>0) print("最初の敵のx座標",a[0].x);

for ... in の使用

for ... in を使って各Actorへに同じ処理を一斉に行うことができます． 

for (e in all(Enemy)) {
   e.die();
}
