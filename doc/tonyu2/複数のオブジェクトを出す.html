複数のオブジェクトを作る

先ほどの例で作ったChara1とChara2を両方出現させるには、別にMainというファイルを作成します。
Main
new Chara1;
new Chara2;

「new ファイル名」と呼び出すと、そのファイル（クラス）のオブジェクトが作られ、そのファイル中に書いてある動作を実行します。

Chara1とChara2 の内容は先程までと一緒です。

Chara1
x=100;
y=100;
while(true){
  x++;
  update();
}
Chara2
x=100;
y=200;
vy=0;
while(true){
  x++;
  y+=vy;
  vy+=0.1;
  if(y>$screenHeight){
    y=$screenHeight;
    vy=-vy;
  }
  update();
}
