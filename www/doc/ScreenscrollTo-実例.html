画面全体をスクロールさせる

  • $Screen.scrollToを用いて画面をスクロールさせることができます．
    • $Screen.setPivotを用いて，キャラクタが画面中央に来るように設定してください

Main

//スクロールしていることをわかりやすくするため、動かないオブジェクトをいくつか置いておく
for (i=0;i<10;i++) {
    new Actor{x:rnd(400),y:rnd(400)};
}
//プレイヤーの作成
new Player;

Player

x=200;y=200;p=$pat_neko;
// $Screen.scrollToの基準点を画面真ん中に設定
$Screen.setPivot($screenWidth/2,$screenHeight/2);
while(true){
    // このキャラクタが画面真ん中に来るように設定
    $Screen.scrollTo(x,y);
    x++;// 横に移動(移動の処理がすでに書いてある場合，この行は省略してください)
    update();
}
