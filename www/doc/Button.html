Buttonクラス

ボタンを作成します。

フィールド

  • top
    • 上位置
  • text
    • ボタンに書く文字
  • left
    • 左位置（デフォルトは50）
  • width
    • 幅(デフォルトは画面幅-100)
  • height
    • 高さ（デフォルトは50)
  • padding
    • 文字とボタン枠の間隔(デフォルトは5)
    • 文字の大きさはpaddingとheightから自動的に計算されます
  • fillStyle
    • ボタンの塗りつぶし色（デフォルトはrgb(179,255,142)）
  • strokeStyle
    • ボタンの枠色(デフォルトは黒）
  • clickedStyle
    • ボタンをクリックしたときの枠色（デフォルトは白）
  • onClick
    • クリック時の動作（待機不能モードで実行）
  • clicked
    • クリックされた瞬間は1、押し続けられている間は2以上、それ以外は0
  • key
    • 省略可．getkeyと同じ方式でキーの名前を指定すると，指定されたキーが押されたときにもクリックされたと見なします．

※ボタンが作成されるレイヤーはデフォルトでは$frontLayerです．それ以外のレイヤーに配置するときはlayerを指定してください．

※色には文字列またはColorオブジェクトを指定します．

メソッド

  • Actorで使えるメソッドが利用できます(dieなど)
