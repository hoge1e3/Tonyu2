乱数を使う

画面の上からランダムにオブジェクトを降らせる

  • オブジェクトの出現位置のx座標を乱数を使って指定します．
    • 乱数の範囲は0から$screenWidth(画面幅のピクセル数)未満の範囲にします．
  • オブジェクトを生成する処理を定期的に行います．

Main

c=0;//経過時間（フレーム数）を数える変数
while(true) {
    c++;
    if (c%30==0) {//30フレームに1回
        //Enemyを出現
        new Enemy{x:rnd($screenWidth),y:0,p:1};
    }
    update();
}

Enemy
