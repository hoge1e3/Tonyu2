BaseActor

getCrashRect

当たり判定に使う矩形領域を返します．
crashTo，crashTo1，allCrash，findTouchで用いられます

書式

getCrashRect()

このオブジェクトの当たり判定に使う矩形領域を返します．

戻り値をrとすると：

  • (r.x,r.y)が矩形領域の中心位置です（左上ではありません）
  • r.widthとr.heightが，矩形領域の幅と高さです．

オーバライド

デフォルトのgetCrashRectは，画像の大きさ*crashScaleを基準に当たり判定を計算しますが，
getCrashRectをオーバーライドすると，当たり判定の大きさを変更できます（crashScaleを明示的に書かない場合，crashScaleの値は無視されるようになります）．

次の例では，当たり判定の大きさを5*5に設定しています．

\getCrashRect() {
   return {x,y,width:5, height:5};
}
