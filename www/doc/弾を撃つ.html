用途別リファレンス

オブジェクトを動かす

弾を撃つ

  • キーボードなどの動作に反応してオブジェクトを生成します
  • 弾が出現する位置は，弾を撃つオブジェクトと同じ位置になるようにします．

Player

while(true) {
//スペースキーを押すとBulletオブジェクトが出現する。
  if(getkey("space")==1){
    // このオブジェクト（Player）と同じ位置にBulletオブジェクトを出現させる※
    new Bullet{x=x, y=y, p=25};
  }
  update();
}

Bullet

//画面上端に到達するまで上に移動していき、上端に到達すると消える。
while(y>0){
  y=y-4;
  update();
}
die();

※上のPlayerクラスの

new Bullet{x=x, y=y, p=25};

は，次のように省略して書くことができます．

//オブジェクトへのパラメタ内で x と書くと x=x と等価
new Bullet{x, y, p=25};
