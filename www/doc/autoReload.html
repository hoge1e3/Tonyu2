Boot

$Boot.autoReload

実行中にファイルが変更され，自動的にコンパイルが終わった後の動作を指定します．

  • trueを指定するとメインページ（最後に実行したクラス）を読み込み直します
  • 関数を指定すると，その関数を実行します．

例

Main

//一度実行して，実行したままこの中身を書き換え，保存(Ctrl+S)してみましょう．
$Boot.autoReload=true;
x=100;
y=150;
while(true) {
    x+=2;//このへんの数値をいじってみましょう
    y+=1;//このへんの数値をいじってみましょう
    if (x<0) x=$screenWidth;
    if (y<0) y=$screenHeight;
    if (x>$screenWidth) x=0;
    if (y>$screenHeight) y=0;
    update();
}
