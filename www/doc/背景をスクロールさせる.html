背景をスクロールさせる

背景の表示を参考に画像を用意してください．

Main

//画像の縦と横の値を入れる
$Screen.resize(500,500);
new Bg{x:$screenWidth/2,y:$screenHeight/2,p:$pat_bg,zOrder:1};
new Bg{x:$screenWidth/2,y:-$screenHeight/2,p:$pat_bg,zOrder:1};

Bg


while(true){
    //画像を下に移動
    y+=10;
    //画像が画面から出たら
    if(y-$screenHeight/2>=$screenHeight){
        //画像を上に戻す
        y=0-$screenHeight/2;
    }    
    update();
}

このプログラムでは、

  • 背景の画像を縦に二枚表示させ，
  • それぞれ下方向に移動させ
  • 画面外に画像が出たら、その画像を上に戻す

という処理によって，連続した画像が流れているように表示しています．
