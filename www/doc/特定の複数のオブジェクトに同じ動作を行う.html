オブジェクトを表示する

特定のクラスの複数のオブジェクトに同じ動作を行う

for(c in all(クラス名)) {...処理...} を用いると，指定したクラスのオブジェクトのすべてに同じ処理を行うことができます．

Main

new Chara1{x:100, y:100};
new Chara1{x:200, y:100};
while(true){
    // 画面内にいるすべてのChara1クラスのオブジェクトに対して繰り返す
    for (c in all(Chara1)) {
        // 各Chara1オブジェクトをcに代入しながら繰り返し
        c.x++;//cを1ピクセル右に移動させる
    }
    update();
}
