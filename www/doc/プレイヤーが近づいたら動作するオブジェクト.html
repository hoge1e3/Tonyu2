用途別リファレンス

プレイヤーが近づいたら動作するオブジェクト

  • withinメソッドを使って，プレイヤーとの距離を計測します
  • 指定した距離に近づいたら行動を開始します

Main

$player=new Player{x=100, y=200};
new Enemy{x=200, y=300};
new Enemy{x=250, y=250};

Enemy

//$playerが近づくまで待つ処理
while(true) {
    if (within($player,100)) {//プレイヤーとの距離が100以内なら
        break;// 待つ処理を抜ける
    }
    update();
}
//近づいたときにとる行動
while(true) {
    x-=10;
    update();
}

Player

//これ以外の動作でもよい
pad=new APad;
while(true){
    x+=pad.vx*5;
    y+=pad.vy*5;
    update();
}
