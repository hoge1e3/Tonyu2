file

FS.listFiles メソッド

ディレクトリ直下のすべてのファイルを返します．

書式

fs.listFiles()

fsがあらわすディレクトリ直下のすべてのファイルまたはディレクトリを，fileオブジェクトの配列で返します．

例

// filesフォルダのファイル名一覧を表示
dir=file("./");
for (let n of dir.listFiles()) {
    print(n.name());
}

// filesフォルダ内のサブフォルダ data/ のファイル名一覧を表示
// (data/フォルダがない場合，dir.listFiles()は空の配列になり，何も表示されない）
dir=file("data/");
for (let n of dir.listFiles()) {
    print(n.name());
}
