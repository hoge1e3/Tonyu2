ボタンを使う

ボタンが押されたとき，複数フレームにまたがる動作を行なう

ボタンが押されたときに，parallelメソッドで別のメソッドを並列実行させます． 

//  クリックされたときは、下に定義したstartWalkメソッドを呼ぶ
$b=new Button{top:300,text:"Click Me",onClick:startWalk};
x=100;
y=100;

function startWalk() {
//walkメソッドを並列実行する
    parallel(walk);
}
// walkメソッドの定義
function walk() {
    for(i=0;i<10;i++) {
       x+=1;
       update();
    }
}
