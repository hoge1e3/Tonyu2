オブジェクトをドラッグする

  • findTouchメソッドを用いて，オブジェクトがタッチされたかを判定する
  • dragメソッドを自分で定義し，タッチされている間，ドラッグ中の移動の処理を行なう 

x=100;y=200;
while(true) {
    t=findTouch();
    if (t) {
        drag(t);
    }
    update();   
}
\drag(t) {
    while(!t.ended) {
        x+=t.vx;
        y+=t.vy;
        update();
    }
}
