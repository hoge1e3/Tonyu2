用途別リファレンス

画面外に出たかどうかの判定

Main
x=50;
y=50;

while(true) {
    if (screenOut()>0) die();//画面外出ると消える
    if (getkey("up")) y-=10;
    if (getkey("down")) y+=10;
    if (getkey("left")) x-=10;
    if (getkey("right")) x+=10;
    update();
}

これは、画面外出るとオブジェクトが消える

「screenOut()」は、画面外に出ていない場合は0、 画面外に出ている場合は0より大きい値を返す。 その値は x方向にはみだしたドット数＋y方向にはみだしたドット数
