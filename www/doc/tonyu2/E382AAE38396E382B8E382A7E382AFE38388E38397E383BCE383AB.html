オブジェクトプール

オブジェクトプールは、dieメソッドで消したオブジェクトを、 別のオブジェクトとして再利用するための仕組みです。 オブジェクトを頻繁に出現・消滅させる場合、メモリの使用効率が上がる場合があります。

オブジェクトプールを利用するには

オブジェクトプールを利用するかどうかはクラス毎に設定します。 ここではChara1にオブジェクトプールを利用する例を示します。

  • オブジェクトが生成されたときにuseObjectPoolをtrueを設定します。

Chara1
useObjectPool=true;
for(i=0;i<10;i++) {
   x+=10;
   update();
}
die();

  • Chara1を生成するときには、appearメソッドを用います。

Main
while(true){
   appear(Chara1) {x:rnd(400),y:rnd(400)};
   update();
}
