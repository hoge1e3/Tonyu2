api

$customLoading

「ウィンドウ」→「プロジェクトオプション」→「グローバル変数」にて$customLoading をtrueにすると，
画像・音声の読み込み完了前にプログラムを開始することができます．主にローディング画面を
作成するのに使います．

  • 読み込みが完了すると$Boot.loadedAll がtrueになります．
  • 読み込みが完了するまでは，次のことは行えません
    • キャラクタパターンを利用した表示
    • $soundの利用

例

//ロード中
x=$screenHeight/2;
y=200;
size=30;
text="Loading...";
while(!$Boot.loadedAll) {
    y=200-sin($frameCount*5)*20;
    update();
}
//ロード完了（これ以降，キャラクタパターンや$soundが利用可能）
text="";
