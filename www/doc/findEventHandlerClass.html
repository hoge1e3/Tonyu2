EventMod

findEventHandlerClass

onメソッドで指定されたイベント名に対応したEventHandlerクラスを返します．
オーバーライドして使用してください．

\findEventHandlerClass(type) {//type:string for event name
    if (type==="myEvent") return MyHandler;// class which extends EventHandler
    return super.findEventHandlerClass(type);
}

上のように設定しておくと，on("myEvent", 処理)などを実行したときに，
MyHandlerクラスを使ってイベントを処理します．
