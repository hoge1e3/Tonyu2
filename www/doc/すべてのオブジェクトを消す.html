オブジェクトを消す

すべてのオブジェクトを消す

  • all().die()...自身以外のオブジェクトをすべて消します
  • all(クラス名).die()...指定されたクラスのオブジェクト（自身以外）をすべて消します

Main

x=$screenWidth/2;
y=$screenHeight/2;
new Enemy{x:100, y:100, p:4};
new Enemy{x:100, y:200, p:4};
new Enemy{x:100, y:300, p:4};
while(true){
    if(getkey("space")==1){//スペースキーを押すと
        all(Enemy).die();//Enemyクラスのオブジェクトが消える。
        die();
    }
    update();
}
