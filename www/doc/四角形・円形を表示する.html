用途別リファレンス

四角形・円形を表示する

画像を使用することなく，四角形や円形をキャラクタとして表示することができます．

四角形の表示

次の値を設定することで四角形を表示します．

  • fillStyle 四角形の色
  • width 幅
  • screenHeight 高さ

Main

new Actor{x=100,y=200,width=300,height=50,fillStyle="red"};

円形の表示

次の値を設定することで円形を表示します．

  • fillStyle 円の色
  • radius 円の半径

Main

new Actor{x=100,y=200,radius=20,fillStyle="yellow"};

物理オブジェクトの場合

物理オブジェクトでも同様に円形や四角形を表示できます．また，衝突判定の大きさも図形の形状にあわせて変化します．

Main

new BodyActor{x=100,y=100,radius=20,fillStyle="green"};
new BodyActor{x=100,y=300,width=100,height=10,rotation:20,fillStyle="#48f",isStatic=true};
