出現するオブジェクトの個数を制限する

確率でオブジェクトを出現させる，弾を撃つ などでオブジェクトを出現させるときに，同時に出現する個数を制限したいときはall(クラス名).lengthでオブジェクトの出現している個数を調べ，決められた数より少ない場合だけ出現させるようにします．

//プログラムの一部．他の部分は参考リンクを参照
  if(rnd(100)<5 && all(Chara1).length<10 ){ // 同時に10個まで出現する
    new Chara1{x:rnd($screenWidth),y:0,p:1};
  }
