画面外に出たかどうかの判定

screenOut()メソッドは、画面外に出ていない場合は0、 画面外に出ている場合は0より大きい値を返します。

これを利用して，画面外に出たらオブジェクトを消す処理を書くことができます．

Main

x=50;
y=50;
while(true) {
    x+=3;
    if (screenOut()>0) die();//画面外に出ると消える
    update();
}

参考

  • 自分自身を消す
  • 他のオブジェクトを消す
  • 当たり判定で当たったオブジェクトを消す
    • 当たり判定を変更する
  • 画面外に出たオブジェクトを消す
  • すべてのオブジェクトを消す
