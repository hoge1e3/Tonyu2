ドラッグやスワイプした量に応じてオブジェクトを動かす

  • $touches[0].vxと $touches[0].vy にドラッグまたはスワイプしたときの移動量が格納されます
  • これらの値をxやyに加算することで移動量に応じてオブジェクトが移動します．
    • ゲーム画面のどこをタッチしても移動できます（オブジェクトの近くをタッチする必要はない）

Main

x=100;y=200;
while(true) {
    x+=$touches[0].vx;
    y+=$touches[0].vy;
    update();   
}
